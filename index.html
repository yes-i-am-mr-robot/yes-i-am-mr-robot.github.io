Currently

Effective retrieval of Web Object

Goal of the project is to reduce the average page access time and thereby save the bandwidth consumption of the server.
Links :  



      The number of Internet users worldwide has skyrocketed since the birth of the World Wide Web. The entire world is migrating towards the Internet. Leave about connecting computers, laptops and smartphones with the internet, now there is a multitude of smart devices connected to the internet and each other. Starting from home appliances to big industrial machinery, everything is become smart. At the root of this exponential growth comes the problem of web latency, higher internet traffic, and more bandwidth requirements.

      To address the issue mentioned above, we propose a novel idea of implementing Prefetching and Caching algorithms in a Clustered Network which will enable all users in a particular cluster to make use of prefetched and cached web objects from all other users.

  • Clustering is done using the BIRCH algorithm, which clusters the users based on the preferences.
  • Caching is done based on the frequency of the requests, as against the LRU method, thereby attempting at increasing the Hit-ratio.
  • Markov model is used to predict the users future requests for prefetching, allowing a faster retrieval of web objects by the users.
